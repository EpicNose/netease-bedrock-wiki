# 传送门零件PortalPart



## 概述

- 描述

    传送门零件

- 成员变量

    | 变量名 | <div style="width: 4em">数据类型</div> | 说明 |
    | :--- | :--- | :--- |
    | portalGateType | int | 传送门类型，0表示单向，1表示双向 |
    | portalGateShape | int | 传送门形状，当前只能是立式方框0 |
    | portalGateWidth | int | 传送门宽度 |
    | portalGateHeight | int | 传送门高度 |
