## 贡献文档

如果你发现了一些小错误，或者想要补充一些新的教程到我们的官网，那我们超级欢迎！

你可以有两种方式参与进来：

1. 直接在本仓库中提交pr，在项目中留下你的大名！
2. 或者加入我们的文档共创群，并将你的文档提交给管理员

提交的文档会经过审核，最终以每两周一次的频率更新到我们的官网中
